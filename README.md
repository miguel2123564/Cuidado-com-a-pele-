  <section id="dicas">
    <h2>Dicas para uma pele saudável</h2>
    <p>Siga estas dicas simples para manter sua pele saudável e radiante:</p>
    <ul>
      <li>Lave o rosto duas vezes ao dia com um sabonete adequado ao seu tipo de pele.</li>
      <li>Use protetor solar diariamente, mesmo em dias nublados.</li>
      <li>Hidrate sua pele com produtos específicos para suas necessidades.</li>
      <li>Evite tocar no rosto com as mãos sujas.</li>
      <li>Beba bastante água para manter a hidratação de dentro para fora.</li>
      <li>Inclua frutas, vegetais e alimentos ricos em antioxidantes na dieta.</li>
    </ul>
  </section>

  <section id="transforme-se">
    <h2>Como mudar sua aparência com cuidados faciais?</h2>
    <p>Transformar sua pele não acontece da noite para o dia, mas com consistência, é possível notar grandes diferenças. Aqui estão algumas sugestões de rotina para iniciar agora:</p>
    <h3>1. Manhã</h3>
    <ul>
      <li>Limpeza facial suave.</li>
      <li>Aplicação de vitamina C para luminosidade.</li>
      <li>Protetor solar com FPS 30 ou mais.</li>
    </ul>
    <h3>2. Noite</h3>
    <ul>
      <li>Remova a maquiagem completamente.</li>
      <li>Use um tônico para equilibrar o pH da pele.</li>
      <li>Hidratante noturno ou creme antienvelhecimento.</li>
    </ul>
